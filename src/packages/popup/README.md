# popup 组件



## props
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| ---- | ---- | ---- | ---- | ---- |
| v-model | 是否可见 | Boolean | / | false |
| position | 弹出方位 | String | full/center/top/bottom/left/right | / |
| show-close | 显示关闭按钮 | Boolean | / | false |
| show-mask | 显示遮罩层 | Boolean | / | true |
| mask-clickable | 遮罩层是否可点击（关闭） | Boolean | / | true |

## events
| 事件名 | 说明 | 回调参数 | 返回值 |
| ---- | ---- | ---- | ---- |
| close | 关闭当前弹层 | / | / |

## slots
| name | 说明 |
| ---- | ---- |
| / | 弹层的内容区域 |
